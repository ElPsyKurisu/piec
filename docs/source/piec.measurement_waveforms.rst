Discrete Waveform
------------------------------
currentmodule:: piec.measurement_waveforms.discrete_waveform

This module handles the creation of a discrete waveform object that can be used to create a waveform for 
a single pulse waveform measurement. Child classes of this object can be used to create different types of waveforms.
Currently, the implimented specific waveforms are :class:'HysteresisLoop' and :class:'ThreePulsePund'.
